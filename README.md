## Справочники

### 1. Горные вершины (`mountain_peaks`)
Содержит информацию о горных вершинах мира.

| Поле              | Тип данных     | Описание                          |
|-------------------|---------------|-----------------------------------|
| name              | TEXT          | Название вершины                  |
| mountain_range    | TEXT          | Горный хребет                     |
| country           | TEXT          | Страна                            |
| height_m          | INTEGER       | Высота над уровнем моря, м        |
| difficulty_rating | DECIMAL(3,1)  | Сложность восхождения (1.0–10.0)  |
| first_ascent_date | DATE          | Дата первого восхождения          |
| description       | TEXT          | Описание (многострочное)          |

### 2. Альпинистские экспедиции (`mountaineering_expeditions`)
Содержит информацию об экспедициях к вершинам.

| Поле         | Тип данных     | Описание                                        |
|--------------|---------------|-------------------------------------------------|
| name         | TEXT          | Название экспедиции                             |
| peak_id      | INTEGER (FK)  | Вершина — ссылка на `mountain_peaks`            |
| start_date   | DATE          | Дата начала экспедиции                          |
| duration_days| INTEGER       | Длительность в днях                             |
| team_size    | INTEGER       | Количество участников                           |
| budget_usd   | DECIMAL(12,2) | Бюджет экспедиции в USD                         |
| description  | TEXT          | Описание маршрута и целей (многострочное)       |

---

## Зависимость справочников

Поле `peak_id` в справочнике **Экспедиции** является внешним ключом,
ссылающимся на запись в справочнике **Горные вершины**.
Одна вершина может фигурировать в нескольких экспедициях.
При удалении вершины из справочника данные об экспедициях сохраняются.

## Покрытие типов данных

| Тип                 | Поля                                            |
|---------------------|-------------------------------------------------|
| Текст (строка)      | name, mountain_range, country                   |
| Текст (многострочный)| description (оба справочника)                  |
| Дата                | first_ascent_date, start_date                   |
| Целое число         | height_m, duration_days, team_size              |
| Дробное число       | difficulty_rating (DECIMAL 3,1), budget_usd (DECIMAL 12,2) |
| Внешний ключ        | peak_id → mountain_peaks.id                     |
